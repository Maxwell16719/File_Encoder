# EncodeFilesPython
Encode your files, really hard to decrypt if you don't know the password.


A few things:

You will need to have python downloaded.
Run the command in terminal:

pip install zstandard cryptography

if that doesn't work use:

pip install zstandard cryptography cffi

Then run: pip install --upgrade pip

How to use:

Press run, and then it is self explanatory:
When pasting a file directory it is okay if it has quotations on each side.
ex: "C:\Users\JohnSmith\Downloads\Bodacious Elzing (1).3mf"


Any files (ex: pdf, mp4, mp3, wav, ZIP), regular folder DO NOT WORK ONLY ZIP FOLDER DO,

MUST BE RUN IS VS STUDIO APP!!!

Thank you for using the encoder.
